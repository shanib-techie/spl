# 💸 SplitSmart CLI – Expense Splitter for Groups

A simple yet powerful **Command Line Interface (CLI) tool** to manage and split expenses among a group of people.

Built to solve real-life confusion when multiple people are spending money in a group and need a clear settlement.

---

## 🚀 Why I Built This

During group expenses, I noticed one major problem:

* Everyone pays randomly
* No one tracks properly
* Final settlement becomes confusing and messy

So instead of relying on manual calculations or apps, I built this tool to:

* Track who paid
* Split expenses intelligently
* Generate final settlements automatically

---

## ⚙️ Features

✅ Add multiple people dynamically
✅ Track transactions person-by-person
✅ Split expenses:

* Among all participants
* Among selected individuals

✅ Automatic balance calculation
✅ Final settlement output (who pays whom)
✅ Clean and simple CLI interaction

---

## 🧠 How It Works

1. Enter number of people and their names
2. Add transactions:

   * Who paid
   * How much
   * Split type (`all` or `specific`)
3. System updates balances in real-time
4. Final settlement shows optimized transactions
